# 「Walico」UI デザイン・配色設定

あなたは割り勘 PWA「Walico（ワリコ）」の UI をデザインします。
デザインコンセプトは **「Smart Mint（スマートミント）」** です。
清潔感があり、ストレスフリーで、動作が軽快に見えるデザインを目指してください。

## 配色パレット (Tailwind CSS)

以下のカラーマッピングを厳守して実装してください。

### 1. プライマリーカラー（ブランドカラー）

- **Primary (Main Action):** `emerald-500` (#10B981)
  - 用途: メインボタン（撮影・確定）、スライダー、アクティブ状態の強調。
- **Primary Hover:** `emerald-600` (#059669)
  - 用途: ボタン等のホバー/タップ時の状態。
- **Primary Light (Backgrounds):** `emerald-50` (#ECFDF5)
  - 用途: 薄い背景色、リスト選択時のハイライト。

### 2. ベースカラー（構造）

- **Background:** `white` (#FFFFFF)
  - 清潔で広々とした印象にするため、基本は白背景。
- **Surface/Cards:** `white` または `gray-50` (#F9FAFB)
  - 境界線（Border）を多用せず、薄い影（Shadow）で階層を表現する。

### 3. タイポグラフィ（文字色）

- **Text Main:** `gray-800` (#1F2937)
  - 可読性を高めるため、真っ黒ではなく濃いグレーを使用。
- **Text Muted:** `gray-500` (#6B7280)
  - 補足情報（日時、ヘルプテキストなど）に使用。

### 4. 機能色

- **Accent/Destructive:** `rose-500` (#F43F5E)
  - 「削除」「キャンセル」などの破壊的アクションにのみ限定して使用。
- **Borders/Dividers:** `gray-200` (#E5E7EB)
  - 区切り線は極力薄くする。

## デザインシステム・ルール

1.  **丸み (Radius):** `rounded-2xl` や `rounded-3xl` を多用し、親しみやすくモダンな印象にする（角張らせない）。
2.  **影 (Shadow):** `shadow-sm` や `shadow-md` を使い、画面下部の固定ボタンやカードを浮き上がらせる。
3.  **余白 (Spacing):** モバイル操作を前提とし、タップ領域を確保するため `p-4` や `gap-4` などのゆとりある余白設定を行う。
